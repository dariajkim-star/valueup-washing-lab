# v2 백로그 — CX 분석 기반 우선순위

> 작성 2026-07-16. 근거: [cx-analysis](../cx-analysis-2026-07-16.md) (여정 지도·HEART·IPA 3방법론 수렴).
> v1은 "정직성 프레임"으로 마감 완료(리드 결정, README 한계 §1·§2) — 이 문서는 그 결정을 뒤집는 것이 아니라, **v2 착수 시 무엇부터 할지**의 우선순위 기록이다. 개별 기술 부채의 전체 목록은 [deferred-work](deferred-work.md).

## 우선순위 산정 방식

IPA 사분면 기준: **집중 개선**(중요도 높음 × 성과 낮음) > 강점 유지 보강 > 저순위. 세 방법론이 독립적으로 같은 근본 원인(UJ-1 Task Success 3.0 = 여정 3단계 감정 꺾임 = IPA 집중 개선 구역)을 가리킨 항목을 P1으로 승격.

## P1 — 집중 개선 (IPA: 중요도 4.0+ / 성과 2.5 이하)

### P1-1. execution_score 커버리지 확대 (IPA 성과 1.5 / 중요도 4.5)

- **현상**: 실데이터 33종목 중 non-null 1종목(기아). UJ-1 3단계(워싱 랭킹)의 체감 완성도를 직접 제약.
- **원인 분해** (deferred-work 기존 항목과 매핑):
  - 배당 커버리지 병목 → **주주환원율 별도 필드** (deferred: payout 커버리지 병목)
  - 목표 필드별 게이팅 → target_roe 없이 배당·자사주만으로도 부분 점수 산출 가능한지 산식 재검토 (SM-C1 "억지 추정 금지"와 충돌하지 않는 범위에서)
- **완료 기준**: non-null 종목 비율 3%→30%+ (유니버스 33종목 기준 10종목+), washing_flag 판정 가능 종목 19→25+.
- **선행 조건**: 없음 (v2 첫 스토리 후보).

### P1-2. 밸류업 목표 파싱 정확도 (IPA 성과 2.5 / 중요도 4.0)

- **현상**: 파서 튜닝으로 24%→42% 개선 후 수확체감 판단으로 v1 마감. 자유서식(범위·서술형) 공시가 원인.
- **접근**: 정규식 추가 튜닝이 아니라 **수동 태깅 보완 레이어** — 파싱 실패 공시에 대해 target 값을 수기 입력하는 관리 테이블(원문 verbatim 보존 + 태깅 근거). PRD Open Question #1(범위→중앙값? 하한?)의 정규화 규칙 확정이 선행.
- **완료 기준**: 파싱+태깅 합산 커버리지 42%→80%+, 태깅 레코드에 원문 근거 필수(정직성 원칙 유지).
- **선행 조건**: PRD OQ-1 정규화 규칙 리드 확정.

## 하지 않을 것 (IPA 과잉 주의 — 명시적 보류)

- **매크로 스코어 가중 확대·매크로 스트립 고도화** (성과 2.5/중요도 4.0이지만 개선 방향이 "가중치 상향"이 아님): 매크로가 M&A 점수에서 15%로 체감이 낮은 것은 설계 의도(종목 변별력은 나머지 85%). 스트립 시각 고도화는 유니버스 확대 전까지 투자 대비 효과 낮음.
